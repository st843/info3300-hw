A: The marks used in this visualization are dog symbols.

B:

  • The popularity data on the y-axis are quantitative.
  • The AKC Dog group data are nominal.

C:

  • Popularity on the y-axis uses vertical aligned position as a visual channel.
  • "Our Data Score" on the x-axis uses horizontal aligned position as a visual channel.
  • AKC Dog Group uses color hue as a visual channel.

D: I think this design choice was effective, because while there was a tradeoff for visual clutter and imprecise locations on the graph, the dog outlines are a compelling visual for the user and uniquely depict the visual channel of dog shapes that would otherwise be absent from the visualization if simple circles were used instead.

A: The marks used in this visualization are neighborhoods.

B:

  • The breed data are nominal.
  • The relative prevalence of breed data are quantitative.

C:

  • The visual channel used for breed is the text labels near each map.
  • The visual channel used for relative prevalence is color hue and luminosity.

D: I believe this warm color scale is effective, because since there is a redundancy in the visual channels such that both hue and luminosity are utilized, this might help those who have color vision deficiencies still see differences between neighborhoods, and also help non-visually impaired people better recognize these differences as well.

A. This type of visualization is called a chord diagram.

B.

  • The visual marks corresponding to the nodes are the curved bands around the edge of the circle.
  • The visual marks correspodning to the edges are the ribbons that connect nodes around the circle.

C. One advantage of using the chord diagram is that especially if there is not an excessive amount of data, the graph can be a visually enticing way for users to trace connections between nodes. One potential drawback of using the chord diagram is that depending on the position of the nodes around the circle, certain edges, like the ones that cross through the middle of the circle, are more visually salient than others, such as those that connect nodes directly adjacent to one another.
